# simple-image-classifier
Simple image classifier microservice using tensorflow

#### PREREQUISITES FOR LOCAL RUN
- A linux system
- python >= 3.5

#### QUICKSTART GUIDE ==
Go to the app folder:
```
cd app/
pip3 install -r requirements.txt
python3 app.py
```

It will listen on localhost:8080
